# Golden Suites

A static website for a hotel/resort landing page. The site includes sections for slideshow hero images, services, rooms, and customer testimonials.

## Project Structure

- `index.html` — Main page markup
- `style/index.css` — Styling for the site
- `pictures/rooms/` — Room images
- `pictures/services/` — Service images
- `pictures/slideshow/` — Hero slideshow images
- `pictures/customers/` — Customer testimonial images
- `fontawesome/` — Font Awesome icon assets

## Notes

- Image paths in `index.html` were updated to use the new `pictures/` folder structure.
- Keep image filenames and subfolder names consistent with the HTML references.

## Usage

Open `index.html` in a web browser to view the page locally.
